4.1 dplyr 包

dplyr 包是R语言中用于数据操作和数据变换的重要包，它提供了一组简单而一致的函数，用于对数据框进行筛选、变换、汇总和可视化等操作。以下是对dplyr包的全面总结，包括其主要函数和示例用法

4.1.1 主要函数：

dplyr包提供了一系列主要函数，用于数据操作和变换。

  1. filter()：用于筛选行，根据指定条件保留符合条件的观测值。
filtered_data <- filter(data_frame, condition)
  1. select()：用于选择列，从数据框中选择指定的列。
selected_data <- select(data_frame, column1, column2)
  1. mutate()：用于创建新列或修改现有列。
mutated_data <- mutate(data_frame, new_column = expression)
  1. arrange()：用于对数据按照指定列的值进行排序。
sorted_data <- arrange(data_frame, column_to_sort)
  1. summarize()：用于对数据进行汇总，计算统计量。
summary_stats <- summarize(data_frame, stat = aggregation_function(column))
  1. group_by()：用于分组数据，为后续汇总操作创建组。
grouped_data <- group_by(data_frame, group_column)
  1. join()：用于连接多个数据框。
merged_data <- left_join(data_frame1, data_frame2, by = "key_column")

4.1.2 案例

下面来看一个简答的例子

# 加载dplyr包
library(dplyr)
## 
## Attaching package: 'dplyr'
## The following objects are masked from 'package:stats':
## 
##     filter, lag
## The following objects are masked from 'package:base':
## 
##     intersect, setdiff, setequal, union
# 创建示例数据框
data_frame <- data.frame(
  ID = 1:5,
  Name = c("Alice", "Bob", "Charlie", "David", "Eve"),
  Age = c(25, 30, 22, 28, 32)
)

# 1. 筛选年龄大于25的观测值
filtered_data <- filter(data_frame, Age > 25)

# 2. 选择ID和Name列
selected_data <- select(data_frame, ID, Name)

# 3. 创建一个新列Age_Group，根据Age列的值分类
mutated_data <- mutate(data_frame, Age_Group = ifelse(Age > 30, "Old", "Young"))

# 4. 按照Age列进行升序排序
sorted_data <- arrange(data_frame, Age)

# 5. 汇总年龄的均值和标准差
summary_stats <- summarize(data_frame, Mean_Age = mean(Age), SD_Age = sd(Age))

# 6. 根据Age_Group列分组
grouped_data <- group_by(data_frame, Name)

# 7. 连接两个数据框
data_frame2 <- data.frame(ID = 1:5, Score = c(85, 92, 78, 88, 95))
merged_data <- left_join(data_frame, data_frame2, by = "ID")

4.2 tidyr 包

tidyr 包是R语言中用于数据重塑和整理的重要包，它提供了一组函数，用于将数据从宽格式转换为长格式，或者从长格式转换为宽格式。

4.2.1 主要函数：

  1. gather()：用于将宽格式数据转换为长格式。
gathered_data <- gather(data_frame, key = "new_key", value = "new_value", columns_to_gather)
  1. spread()：用于将长格式数据转换为宽格式。
spread_data <- spread(data_frame, key = "new_key", value = "new_value")
  1. separate()：用于将一个列拆分为多个列。
separated_data <- separate(data_frame, col = "column_to_split", into = c("new_col1", "new_col2"), sep = "separator")
  1. unite()：用于将多个列合并为一个列。
united_data <- unite(data_frame, col = "new_column", cols_to_unite, sep = "separator")

4.2.2 例子

下面是使用tidyr包进行数据整理和重塑的示例：

# 加载tidyr包
library(tidyr)

# 创建示例数据框
data_frame <- data.frame(
  ID = 1:3,
  Name = c("Alice", "Bob", "Charlie"),
  Test1_Score = c(95, 88, 92),
  Test2_Score = c(87, 91, 84)
)

# 1. 将宽格式数据转换为长格式
long_data <- gather(data_frame, key = "Test", value = "Score", Test1_Score, Test2_Score)

# 2. 将长格式数据转换为宽格式
wide_data <- spread(long_data, key = "Test", value = "Score")

# 创建示例数据框
data_frame2 <- data.frame(
  ID = 1:3,
  FullName = c("Alice Johnson", "Bob Smith", "Charlie Brown"),
  Age = c(25, 30, 22)
)

# 3. 将FullName列拆分为First_Name和Last_Name列
split_data <- separate(data_frame2, col = FullName, into = c("First_Name", "Last_Name"), sep = " ")

# 4. 将First_Name和Last_Name列合并为FullName列
united_data <- unite(split_data, col = "FullName", First_Name, Last_Name, sep = " ")

4.3 stringr

stringr 包是R语言中用于字符串处理的包，它提供了一组简单而一致的函数，用于对字符串进行操作、提取信息和进行模式匹配等操作。

4.3.1 主要函数：

  1. str_length()：用于计算字符串的长度。
library(stringr)
string_length <- str_length("Hello, world!")
  1. str_c()：用于连接多个字符串。
concatenated_string <- str_c("Hello", ", ", "world!")
  1. str_sub()：用于截取字符串的子串。
substring <- str_sub("Hello, world!", start = 1, end = 5)
  1. str_replace()：用于替换字符串中的指定子串。
replaced_string <- str_replace("Hello, world!", "world", "there")
  1. str_detect()：用于检测字符串中是否包含特定模式
contains_pattern <- str_detect("Hello, world!", "world")
  1. str_split()：用于将字符串分割为子字符串
split_strings <- str_split("apple,banana,cherry", ",")

4.3.2 例子

# 加载stringr包
library(stringr)

# 1. 计算字符串长度
string_length <- str_length("Hello, world!")

# 2. 连接多个字符串
concatenated_string <- str_c("Hello", ", ", "world!")

# 3. 截取字符串的子串
substring <- str_sub("Hello, world!", start = 1, end = 5)

# 4. 替换字符串中的子串
replaced_string <- str_replace("Hello, world!", "world", "there")

# 5. 检测字符串中是否包含特定模式
contains_pattern <- str_detect("Hello, world!", "world")

# 6. 将字符串分割为子字符串
split_strings <- str_split("apple,banana,cherry", ",")

4.4 lubridate

lubridate 包是R语言中用于日期和时间处理的包，它提供了一组函数，用于解析、操作、格式化和计算日期时间数据。

lubridate包提供了一系列主要函数，用于日期和时间处理。

  1. 日期解析函数：
  • ymd()：解析年-月-日格式的日期。
  • mdy()：解析月-日-年格式的日期。
  • dmy()：解析日-月-年格式的日期。
  1. 日期时间提取函数：
  • year()：提取日期的年份。
  • month()：提取日期的月份。
  • day()：提取日期的日份。
  • hour()：提取时间的小时。
  • minute()：提取时间的分钟。
  • second()：提取时间的秒数。
  1. 日期时间格式化函数：
  • format()：将日期时间格式化为指定的字符串格式。
  1. 日期时间运算函数：
  • duration()：创建时间间隔。
  • interval()：创建时间区间。
# 加载lubridate包
library(lubridate)
## Loading required package: timechange
## 
## Attaching package: 'lubridate'
## The following objects are masked from 'package:base':
## 
##     date, intersect, setdiff, union
# 创建日期对象
date_obj <- ymd("2023-10-30")

# 1. 提取日期的年份、月份和日份
year_value <- year(date_obj)
month_value <- month(date_obj)
day_value <- day(date_obj)

# 2. 格式化日期时间为指定格式
formatted_date <- format(date_obj, "%Y-%m-%d")

# 创建日期时间对象
datetime_obj <- ymd_hms("2023-10-30 12:34:56")

# 3. 提取时间的小时、分钟和秒数
hour_value <- hour(datetime_obj)
minute_value <- minute(datetime_obj)
second_value <- second(datetime_obj)

# 4. 创建时间间隔
time_interval <- duration(hours = 2, minutes = 30)

# 5. 创建时间区间
time_interval <- interval(start = ymd_hms("2023-10-30 12:00:00"), end = ymd_hms("2023-10-30 14:30:00"))

lubridate包提供了方便的函数，用于处理日期和时间数据，使日期解析、提取和计算等操作更加简单和高效。这些函数对于数据分析和时间序列分析非常有用。

4.5 purrr

purrr 包是R语言中用于函数式编程的包，它提供了一组功能强大的工具，用于处理数据、应用函数、映射、过滤和简化编程任务。

purrr包提供了一系列主要函数，用于函数式编程和数据操作。

  1. map()：用于将函数应用于列表、向量或数据框的每个元素，并返回结果列表。

  2. map2()：用于同时遍历两个列表并将函数应用于对应元素。

  3. map_df()：类似于map()，但返回一个数据框。

  4. map_dbl()：类似于map()，但返回一个双精度向量。

  5. map_int()：类似于map()，但返回一个整数向量。

  6. map_lgl()：类似于map()，但返回一个逻辑向量。

  7. map_chr()：类似于map()，但返回一个字符向量。

  8. map_if()：根据条件筛选要应用函数的元素。

# 加载purrr包
library(purrr)

# 创建一个数字列表
numbers_list <- list(1, 2, 3, 4, 5)

# 1. 使用map函数将每个元素加倍
doubled_numbers <- map(numbers_list, ~ .x * 2)

# 2. 使用map2函数将两个列表对应元素相加
list1 <- list(1, 2, 3)
list2 <- list(4, 5, 6)
summed_list <- map2(list1, list2, ~ .x + .y)

# 3. 使用map_df函数将每个元素的平方转换为数据框
squared_numbers <- map_df(numbers_list, ~ data.frame(Square = .x^2))

# 4. 使用map_dbl函数计算每个元素的平均值
average_value <- map_dbl(numbers_list, ~ mean(.x))

# 5. 使用map_lgl函数检查每个元素是否大于3
greater_than_3 <- map_lgl(numbers_list, ~ .x > 3)

# 6. 使用map_chr函数将每个数字转换为字符串
string_numbers <- map_chr(numbers_list, ~ as.character(.x))

4.6 forcats

forcats 包是Tidyverse生态系统中的一个包，用于因子变量的处理和操作。它提供了一组函数，用于重新编码、排序、筛选和操作因子变量。

forcats包提供了一系列主要函数，用于因子变量的处理。

  1. fct_reorder()：重新编码因子变量的水平，以便按照指定的顺序进行排序。

  2. fct_infreq()：将因子变量的水平按照出现频率进行排序，将最常出现的水平排在前面。

  3. fct_relevel()：重新排列因子变量的水平，可以根据新的顺序进行排列。

  4. fct_rev()：反转因子变量的水平顺序。

  5. fct_drop()：删除不使用的因子水平。

  6. fct_expand()：扩展因子变量，添加缺失的水平。

  7. fct_count()：计算每个因子水平的频数。

# 加载forcats包
library(forcats)

# 创建数据框
data <- data.frame(Gender = c("Male", "Female", "Male", "Female", "Male", "Male"))

df <- data.frame(
  City = c("New York", "Los Angeles", "Chicago", "Houston", "Phoenix"),
  Population = c(8537673, 39776830, 2705994, 2320268, 1680992)
)
# 1. 重新编码因子变量的水平以按照字母顺序排序
df$City <- fct_reorder(df$City, df$Population)


# 2. 将因子水平按照出现频率排序
data$Gender <- fct_infreq(data$Gender)

# 3. 重新排列因子水平的顺序
data$Gender <- fct_relevel(data$Gender, "Female", "Male")

# 4. 反转因子水平的顺序
data$Gender <- fct_rev(data$Gender)

# 5. 删除不使用的因子水平
data$Gender <- fct_drop(data$Gender)

# 6. 扩展因子变量，添加缺失的水平
data$Gender <- fct_expand(data$Gender, "Unknown")

# 7. 计算每个因子水平的频数
level_counts <- fct_count(data$Gender)

4.7 httr

httr 包是R语言中用于进行HTTP请求的包，它提供了一种方便的方式来与Web服务和API进行交互。

httr包提供了一系列主要函数，用于发起HTTP请求和处理响应。

  1. GET()：发起GET请求，获取数据。

  2. POST()：发起POST请求，提交数据。

  3. PUT()：发起PUT请求，更新数据。

  4. DELETE()：发起DELETE请求，删除数据。

  5. content()：从HTTP响应中提取内容。

  6. status_code()：获取HTTP响应的状态代码。

  7. headers()：获取HTTP响应的标头。

  8. add_headers()：添加自定义标头到HTTP请求。

# 加载httr包
library(httr)

# 1. 发起GET请求
response <- GET("https://jsonplaceholder.typicode.com/posts/1")

# 2. 获取HTTP响应的内容
post_data <- content(response, "text")

# 3. 获取HTTP响应的状态代码
status <- status_code(response)

# 4. 添加自定义标头到HTTP请求
custom_headers <- add_headers(Authorization = "Bearer YourToken")
custom_response <- GET("https://api.example.com/data", custom_headers)

# 5. 发起POST请求，提交JSON数据
json_data <- '{"name": "John", "age": 30}'
post_response <- POST("https://api.example.com/users", body = json_data, encode = "json")

# 6. 发起PUT请求，更新数据
update_data <- '{"name": "John Smith"}'
put_response <- PUT("https://api.example.com/users/1", body = update_data, encode = "json")

# 7. 发起DELETE请求，删除数据
delete_response <- DELETE("https://api.example.com/users/1")

4.8 rlang

rlang包提供了一系列主要函数，用于操作表达式和执行高级编程任务。

  1. enexpr()：将参数转换为表达式。
  2. enquo()：将参数转换为quosure，可用于捕捉函数参数。
  3. quo_name()：提取quosure的名称。
  4. eval_tidy()：以一种干净的方式执行表达式。
  5. quo()：创建quosure对象。
  6. expr()：创建表达式对象。
  7. call2()：构建函数调用表达式。
  8. new_function()：创建自定义函数。
  9. dots_list()：创建参数列表。
  10. lang()：构建自定义语言。

这个包对于高级R编程比较重要, 关于这个包更多内容参考:https://rpubs.com/liam/rlang
